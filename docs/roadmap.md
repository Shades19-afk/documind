# DocuMind roadmap

## Phase 1 — Product UX and summarizer-first flow

- Update the landing page and dashboard messaging to emphasize summaries and study tools.
- Promote study package generation after document processing.
- Add export actions for markdown and JSON.
- Remove legacy chat and vector-search code paths.

## Phase 2 — Reliability and production readiness

- Add persistent storage for processed documents and study packages.
- Add authentication and user/workspace scoping.
- Introduce background job processing for PDF uploads.
- Add robust error handling and retry flows.

## Phase 3 — Growth and collaboration

- Add team sharing, comments, and collaboration features.
- Add search and tagging improvements.
- Add usage analytics and subscription controls.
- Add deployment automation and observability.
